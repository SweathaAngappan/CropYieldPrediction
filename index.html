Why log-space R²

Production spans single digits to hundreds of millions in this dataset (coconuts are counted in nuts, cereals in tonnes). Training and scoring directly on raw values lets a handful of extreme rows dominate the loss. The model is trained on log1p(Production), which is the standard fix for a target spanning several orders of magnitude, and R² is reported in that same space — where it actually reflects fit quality across the full range of crops, not just the largest few.

Notes on this deployment

This dashboard ships a lightweight, quantized export of the trained Random Forest (10 trees, depth 7) so it can run client-side with no backend or API calls. The full model in the notebook uses more trees and finer splits for higher accuracy offline — a standard trade-off between deployment footprint and headline accuracy.
